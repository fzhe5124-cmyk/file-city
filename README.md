# File City · 文件城市

将本地文件系统映射为 3D 交互式城市的可视化工具。选择文件夹后，文件结构以赛博朋克风格的未来城市形态呈现——文件夹是摩天大楼，文件是建筑群，颜色代表类型，发光强度代表活跃度。

## 使用

用 Chrome/Edge 打开 `index.html`，点击「选择文件夹」即可。

要求浏览器支持 File System Access API（Chrome 86+, Edge 86+）。

## 操作

| 操作 | 功能 |
|------|------|
| 鼠标拖拽 | 旋转视角 |
| 滚轮 | 缩放 |
| 右键拖拽 | 平移 |
| 左键点击建筑 | 查看详情 + 视角聚焦 |
| 双击建筑 | 视角聚焦 |
| R | 复位摄像机 |
| F | 飞行模式（WASD 移动，鼠标旋转，Q/E 升降） |
| Space | 自动旋转开关 |
| S | 截图保存 |
| UP / DOWN | 搜索结果中切换 |
| ESC | 关闭详情 / 清除搜索 |

## 功能

- **文件扫描**：递归扫描文件夹，支持进度显示和取消
- **城市生成**：树形布局 + 力导向微调，量化分位数映射建筑高度
- **颜色编码**：按扩展名分色，文件夹颜色由子文件颜色混合
- **建筑生长动画**：扫描完成后建筑从地面生长出来
- **时光回溯**：底部滑块按修改时间过滤，发光强度平滑过渡
- **搜索定位**：搜索文件/文件夹，匹配建筑逐个点亮，摄像机弧线飞行
- **自动聚类**：检测文件类型区域并显示区域标签
- **统计面板**：文件/文件夹/总大小/排行榜
- **导出**：截图（带水印）、HTML 报告（含文件类型分布、Top 10 排行榜）

## 技术栈

Three.js r160, CSS2DRenderer, UnrealBloomPass, 零外部依赖。
